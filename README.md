# Zephyr Project 中文文档

## 资源文件描述

本仓库提供的是 Zephyr OS 文档的中文翻译版本，版本号为 v1.6.0。该文档旨在帮助中文开发者更好地理解和使用 Zephyr 操作系统。

该文档涵盖了 Zephyr 操作系统的各个方面，包括但不限于：

- 安全机制
- 应用开发指南
